Real-Time Website Health Monitoring System

Developed a full-stack JavaScript system with WebSocket and Azure PostgreSQL for real-time website status monitoring, enabling users to track multiple URLs, view live status, and analyze performance trends through interactive graphs.

HelpLink – A disaster response and relief mobile application

Developed a disaster response mobile app with real-time reporting, shelter navigation, Google Sign-In, PayPal integration, and an AI Chat Assistant, leveraging AWS for backend services to enhance safety, support, and aid coordination during disasters.

Bujang Western – Food Ordering System

Developed a dual-mode food ordering system (walk-in via table QR and delivery via website) with an integrated kitchen staff module that displays incoming orders with status tracking (Pending, Ready, Complete), supports order cancellation, and provides order history management, with receipts sent automatically via Gmail.
